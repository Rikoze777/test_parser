# test_parser

## Установка

```bash
pip install -r requirements.txt
```

## Использование

Сохранит данные в json файл.

1. Запустить скрипт `bet` версии:
```bash
python3 ol_bet.py
```

2. Запустить скрипт `com` версии:
```bash
python3 ol_bet.py
```
